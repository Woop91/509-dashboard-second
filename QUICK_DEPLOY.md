## üîß 5-Minute Setup

### **Step 1: Get the Files**

**Option A: Build it yourself (recommended)**
```bash
git clone https://github.com/Woop91/509-dashboard.git
cd 509-dashboard
npm run build
# Creates two files:
#   - ConsolidatedDashboard.gs (~790 KB) - main code
#   - ConsolidatedDeveloperTools.gs (~63 KB) - demo tools
```

**Option B: Download directly**
Download both files from the repository:
- `ConsolidatedDashboard.gs` (main code)
- `ConsolidatedDeveloperTools.gs` (demo tools - delete before production)

### **Step 2: Create a New Google Sheet**

1. Go to [sheets.google.com](https://sheets.google.com)
2. Click **"+ Blank"** to create a new spreadsheet
3. Name it **"SEIU 509 Dashboard"**

### **Step 3: Open Apps Script Editor**

1. In your Google Sheet, click **Extensions ‚Üí Apps Script**
2. **Delete all the default code** (select all and delete)

### **Step 4: Paste the Main Code**

1. Open `ConsolidatedDashboard.gs` in your text editor
2. **Select ALL** (Ctrl+A / Cmd+A)
3. **Copy** (Ctrl+C / Cmd+C)
4. **Paste** into the Apps Script editor (replacing Code.gs)
5. **Save** the project (Ctrl+S)

### **Step 4b: Add Developer Tools (for demo data)**

1. In Apps Script, click **+** next to "Files" ‚Üí "Script"
2. Name it `DeveloperTools`
3. Open `ConsolidatedDeveloperTools.gs` in your text editor
4. **Select ALL** and **Copy**
5. **Paste** into the new DeveloperTools.gs file
6. **Save** the project

### **Step 5: Authorize the Script**

1. Select **`CREATE_509_DASHBOARD`** from the function dropdown
2. Click the **‚ñ∂Ô∏è Run** button
3. Click **"Review permissions"** ‚Üí Choose your account
4. Click **"Advanced"** ‚Üí **"Go to 509 Dashboard Scripts (unsafe)"**
5. Click **"Allow"**
6. The script creates all sheets (~10 seconds)

### **Step 6: Refresh the Page**

1. Close the Apps Script editor tab
2. Go back to your Google Sheet
3. **Refresh the page** (F5)
4. You should see custom menus appear

### **Step 7: Seed Test Data (Optional)**

1. Click **üîß Admin ‚Üí üé≠ Demo Data ‚Üí üöÄ Seed All Sample Data**
2. Wait for seeding to complete

## üõ†Ô∏è Troubleshooting

### **Menus don't appear?**
- Refresh the page (F5)
- Run `onOpen()` manually from Apps Script editor

### **Authorization error?**
- Complete Step 5 fully
- Close and reopen the Google Sheet

### **Dashboard shows errors?**
- Run `REPAIR_DASHBOARD()` from Administrator menu
- Run `DIAGNOSE_SETUP()` to check system health
