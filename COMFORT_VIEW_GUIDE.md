## Tips for Visual Comfort Success

### Best Practices

**DO:**
- Start with **Dashboard** (your personalized view with My Cases tab)
- Hide gridlines (cleaner look)
- Use larger fonts if text feels overwhelming
- Scan visual icons, not text
- Check dashboard once in morning, once at lunch
- Set phone reminder to check dashboard

**DON'T:**
- Try to read every number
- Open all tabs at once (overwhelming!)
- Stare at tiny text
- Force yourself to use gridlines if they bother you
- Skip the visual guides

### Focus Techniques

**If You Feel Overwhelmed:**
1. Close all tabs except **Dashboard**
2. Look ONLY at the 4 big numbers
3. Take a breath
4. That's enough!

**If You Need More Detail:**
1. Go to **Dashboard**
2. Find the ONE metric you care about
3. Ignore the rest
4. Done!

**If You Hyperfocus:**
- Set a timer for 5 minutes
- Check dashboard
- When timer goes off, STOP
- Move to next task
